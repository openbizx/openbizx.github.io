What is Openbizx?

Openbizx is PHP framework with unique paradigm: MetaObject. In Openbizx, developers only need use 20% of their time to write php logic code and 80% used to write configuration on XML, PHP array, JSON or YAML.
Openbiz have 3 keyword : Simple, Powerfull and Enterprise.

Simple

Openbizx simplify repeated task and than cretae new task only with configuration. Openbizx also use general idiom, decrease technical jargon and use semantic word.

Powerfull

Openbizx is Rapid Development Tools, with secure in mind and of course great User eXperince design.

Enterprise

Openbizx aimed to build enterprise application, like ERP, CRM, CMS, etc. Openbiz is modern framework that use PHP-FIG Standard and targeted runing well on PHP 7.
